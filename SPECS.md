# This file describes specifications for packages for sbpm
This is reflected in the github.com/rambile/sbpm-test

# Required packages
For listing requiered packages, you need to use "dependencies" file, in which you specify needed packages with `deplist"Package manager"=(your packages here)`

# Setup
Right now setup allows you to write arbitrary bash scripts that will execute when you "get" the package. You can use this to copy files to the desired locations.
